# 概览

* Parca是一个持续性能分析项目.
* 持续性能分析是指系统地对程序进行性能分析（例如 CPU、内存、I/O 等）。
* Parca 会采集、存储这些性能分析数据，并使其能够随时间推移进行查询。

## 功能特性

Parca 的主要特性包括：
* 一个多维数据模型，其中包含一系列由配置文件类型和键值对标识的配置文件
* 一种基于标签选择器的查询语言
* 专为分析数据而设计的查询引擎
* 优化的内置存储
* 支持从目标推送和拉取配置文件
* 目标可通过服务发现或静态配置进行发现
* 基于 eBPF 的超低开销分析器
